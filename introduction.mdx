## Core Directories

- `src/` — Source directory containing all application code
  - `assets/` — Static assets like images, fonts, and icons.
  - `components/` — Reusable React components
    - `github/` — GitHub integration components
    - `login/` — Authentication and user management components
    - `news/` — News display components
    - `stock/` — Stock market components
    - `tabs/` — Tab navigation components
    - `weather/` — Weather display components
  - `redux/` — Redux state management
    - `feature/login` — Login-related Redux slices and actions
    - `store.js` — Redux store configuration
  - `utils/` — Utility functions and helper methods
  - `App.css` — Global CSS styles
  - `App.jsx` — Root React component
  - `index.css` — Entry CSS file
  - `main.jsx` — Application entry point

## Directory Details

### `components/`
Individual UI components that are used throughout the application. Each component is organized by feature:
- `github/` — Repository search and details display
- `login/` — Authentication forms and user management
- `news/` — News articles and category navigation
- `stock/` — Stock charts and market data visualization
- `weather/` — Weather conditions and forecasts

### `redux/`
State management using Redux, organized by features:
- `feature/login` — Login state management and authentication
- `store.js` — Central Redux store configuration
